# Xiaohongshu Paper Shares

这个目录用于整理面向小红书的专业论文图文分享。目标不是追热点，而是把论文讲成对 AI 研究和工程实践都有价值的技术解读。

## 受众

- AI 方向硕博。
- 大厂算法、平台、应用、infra 工程师。
- 对 LLM training、agent、RAG、multimodal、reasoning、evaluation 感兴趣的技术读者。

## 内容标准

- 先讲问题定义，再讲方法。
- 可以使用公式，但要解释每个变量的含义。
- 不做“论文翻译”，要讲出自己的结构化理解。
- 对实验结果要说明它支撑了什么 claim，也要指出没支撑什么。
- 必须有 know-how 和 takeaways，让读者知道这篇论文能怎么用。

## 目录建议

```text
xiaohongshu/
  drafts/       # 未发布草稿
  published/    # 已发布版本
  figures/      # 跨文章复用图
```

单篇草稿建议命名：

```text
drafts/YYYY_short-title.md
```

## 图文建议

- 封面图: 论文核心问题 + 一句话判断，不要只放论文标题。
- 图 1: 问题定义/旧方法痛点。
- 图 2: 方法框架。
- 图 3: 关键公式或训练目标。
- 图 4: 主要实验结果。
- 图 5: know-how / takeaways。
