# Skills

这里放可分享、可复制到 Codex 或 Claude 项目中的 workflow skills。它们不是本机安装目录；如果要让 Codex 自动发现某个 skill，可以把对应子目录复制到 `~/.codex/skills/`。

## 当前 skills

- [know-your-unknowns](know-your-unknowns/SKILL.md): 把 HTML-effectiveness 的 `Know your unknowns` 三阶段工作法整理成可给 Codex / Claude 使用的 agent workflow skill。

## 来源和许可证

`know-your-unknowns` 是基于 HTML-effectiveness gallery 中 `Know your unknowns` 示例的改写和工作流化整理，不包含原项目的 HTML/CSS/JS 源文件。发布时请保留 [THIRD_PARTY_NOTICES.md](know-your-unknowns/THIRD_PARTY_NOTICES.md)。
